# 知识架构师 Agent (Knowledge Architect Agent)

组织经验资产化引擎 - 将一线非结构化素材转化为标准化、可复用的培训资产。

## 项目概述

知识架构师是一个 AI 驱动的应用，帮助培训讲师、课程研发人员和业务专家将录音、文档、课件等素材转化为：
- 演示文稿 (PPT)
- 播客脚本
- 金牌话术清单

## 使用指南

### 1. 基础使用（无需 API Key）
- 直接上传文件，系统会使用模拟数据进行演示
- 体验完整的 UI 流程和交互

### 2. 使用真实 API
1. 点击右上角"设置"按钮
2. 开启"启用真实 API"开关
3. 输入 OpenAI API Key（获取地址：https://platform.openai.com/api-keys）
4. 保存设置
5. 上传文件即可进行真实的文档分析

### 3. 支持的文件格式
- 📄 PDF 文档
- 📝 Word 文档 (.docx)
- 📊 PowerPoint 演示文稿 (.pptx)
- 🎵 音频文件 (.mp3, .wav, .m4a)

### 4. 可用的 API 服务

**Groq Whisper API**（推荐，免费）
- 超快速的语音转文字服务
- 支持多种音频格式
- 免费获取：https://console.groq.com/keys

**OpenAI API**（推荐）
- 支持多模态分析（文档、图片、音频）
- 强大的内容理解和生成能力
- 获取：https://platform.openai.com/api-keys

**其他可选服务**
- PDF.ai：专门的 PDF 解析服务 - https://pdf.ai/parse
- 火山引擎豆包：国内可用的大模型服务 - https://www.volcengine.com/docs/82379/1273558

### 5. 音频文件处理指令

上传音频文件后，可以使用以下指令：
- **总结提炼**：提取录音中的核心内容和关键信息
- **提取话术**：从销售/培训录音中提取金牌话术和脚本
- **生成大纲**：为录音内容生成演示文稿大纲
- **自定义指令**：输入任何自然语言指令，AI 会根据指令处理

### 6. 完整使用流程

#### 方式一：文档生成 PPT
1. 上传 PDF/Word 文档
2. 系统自动分析提取关键信息
3. 选择 PPT 模板和配置
4. 点击"确认生成"
5. 查看生成的 PPT 大纲
6. 与 AI 对话进行调整
7. **点击"导出 PPTX"下载真实的 PPT 文件**

#### 方式二：录音生成 PPT
1. 上传音频文件（.mp3/.wav/.m4a）
2. 输入处理指令（如"提取销售话术"）
3. 系统自动转录并分析
4. 选择 PPT 模板和配置
5. 生成结构化的演示文稿
6. 实时预览和调整
7. **点击"导出 PPTX"下载真实的 PPT 文件**

#### 方式三：录音生成播客（新功能）
1. 上传音频文件（.mp3/.wav/.m4a）
2. 输入"生成播客"或选择播客模式
3. 系统自动转录并分析
4. 选择播客风格（故事版/访谈版/对话版）
5. AI 自动生成播客脚本
6. **点击"播放播客"即可收听（无需 API Key）**
7. 下载播客脚本继续编辑

### 7. PPT 导出功能

**支持的功能：**
- ✅ 生成真实的 .pptx 文件（可在 PowerPoint/WPS 中打开）
- ✅ 10 种专业模板自动应用
- ✅ 智能布局（封面页、内容页、两栏布局）
- ✅ 自动格式化（标题、正文、项目符号）
- ✅ 一键下载到本地

**使用方法：**
1. 完成 PPT 大纲生成
2. 在预览页面点击"导出 PPTX"按钮
3. 文件自动下载到浏览器默认下载目录
4. 使用 PowerPoint、WPS 或 Keynote 打开编辑

### 8. 播客生成功能（内置，无需 API）

**核心特点：**
- ✅ **完全免费**：使用浏览器原生 Web Speech API
- ✅ **无需配置**：不需要任何 API Key
- ✅ **即时播放**：点击即可收听播客
- ✅ **三种风格**：故事版、访谈版、对话版
- ✅ **智能脚本**：AI 自动生成对话内容

**使用方法：**
1. 在首页选择"播客"模式
2. 上传录音或文档
3. 输入需求（如"生成故事版播客"）
4. 选择播客风格和配置
5. 点击"确认并生成播客"
6. 查看生成的播客脚本
7. 点击"播放播客"即可收听
8. 点击"下载脚本"保存文本

**播客风格说明：**
- **故事版**：生动有趣的叙事风格，适合知识分享
- **访谈版**：专业的问答形式，适合深度访谈
- **对话版**：轻松的双人对话，适合日常讨论
